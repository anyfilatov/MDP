Тут будет система взаимодействия в облаке

Для запуска сервера необходимо собрать его с помощью компилятора Qt.
при запуске передать следующие параметры
1. Путь до конфигурационного файла config.json, лежащего в папке CloudApp.
2. Id запускаемого хоста. Для хоста с этим id должна быть запись в конфигурационном файле по ключу "hosts". Для него должны быть указаны ip, port, id. 

Пример конфигурационного файла
"hosts": [
        {"ip" : "169.254.99.4", "port" : 4001, "id" : 1 },
        {"ip" : "169.254.99.4", "port" : 4002, "id" : 2 },
        {"ip" : "169.254.99.4", "port" : 4003, "id" : 3 },
        {"ip" : "169.254.99.4", "port" : 4004, "id" : 4 }
    ],
  "relations": [
    {"parent": 1, "childs": [2, 3] },
    {"parent": 2, "childs": [4] },
    {"parent": 3, "childs": [] },
    {"parent": 4, "childs": [] },
  ],
  "db": [
    {"ip": "169.254.99.6", "port": 2323, "id" : 1},
    {"ip": "169.254.99.6", "port": 2323, "id" : 1}
  ],
  "rb": [
    {"ip": "/home/savinov/edu/mdp/mdp/OrderedData/settings.json", "port": 1, "id" : 1}
  ]
  
  Ключ hosts содержит в себе массив конфигураций хостов. Для каждого указаны ip, port, id.
  Ключ relations содержит в себе отношения хостов. Кто кому ребенок. По конфигу примера будет создано следующее дерево
       (1)
      /   \
    (2)   (3)
      \
      (4)
  Ключ db - это конфигурации для DataBase. В каждом указаны host, port и id. id - обязательный ключ, но он нигде не учавствует.
  Ключ db - это конфигурации для RBTree. В каждом указаны host, port и id. id, port - обязательные ключи,
  но их значения нигде не учавствуют, поэттому могут принимать любые значения.
  ключ ip должен содержать путь до конфигурацинного файла RBTree.
  
  Пример запуска
  ./build/CloudApp ./CloudApp/config.json 1
  
  ./build/CloudApp ./CloudApp/config.json 2
  
  Для того, чтобы отправить запрос, необходимо воспользоваться проектом tests/TestProject (есть проект netbeans). Код, ip, port меняются в коде программы.
  
  Если отсутствует БД, то можно сгенерировать случайные значения. Для этого нужно в файле include/DataBase.h закоментировать строчки 32 и 30. и раскоментировать, с 33 по 38.
